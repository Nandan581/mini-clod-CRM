Mini Cloud CRM 

A simple cloud-based CRM mini project built with **Python (Flask)**, **SQLite**, and **HTML/CSS**.  
Users can add customers and leads, and view a dashboard with basic statistics.

Features

- Add and list customers (name, email, phone, company)
- Add and list leads (customer name, deal value, status)
- Dashboard showing:
  - Total customers
  - Total leads
  - Leads by status (New, Contacted, Won, Lost)
- Uses SQLite database (can be swapped with a cloud DB)
- Ready to deploy on a cloud platform (e.g. Render)

Tech Stack

- Python 3
- Flask
- Flask-SQLAlchemy
- SQLite
- HTML + CSS (Jinja2 templates)
